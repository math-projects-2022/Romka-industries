Инструкция для пользователя.

1) Введите параметры для вычисления интеграла.
2) Введите желаемую точность вычисления.
3) Выберите метод (-ы) вычисления интеграла из предложенных. (Необходимо отметить галочкой нужный(-е) пункт(-ы))
(-По формуле центральных прямоугольников)
(-По формуле трапеций)
(-По формуле Симпсона)
4) Нажмите кнопку "Рассчитать".

Вам будет выведен график, отображенный на центральной панели.
